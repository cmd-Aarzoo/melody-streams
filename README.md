🎵 Melody Streams

A modern, lightweight web application that demonstrates a secure user authentication system combined with a clean and interactive UI.

📌 About the Project

Melody Streams is a web-based platform built to showcase the integration of frontend and backend technologies in a simple yet effective way.
It allows users to sign up, log in, access a personalized profile, and log out securely, while providing a smooth and visually appealing experience.

This project is designed especially for students and beginners to understand real-world web development concepts like authentication, session management, and UI/UX design.

✨ Features
🔐 User Authentication (Signup / Login / Logout)
👤 Personalized Dashboard & Profile
⚡ Asynchronous Requests using Fetch API (No page reload)
🎨 Modern UI (Glassmorphism, animations, parallax effects)
📱 Fully Responsive Design
✅ Form Validation (Client-side + Server-side)
🗂️ XML-based Data Storage
🔄 Session Management for secure access
💬 Real-time error & success messages
🛠️ Tech Stack
Category	Technologies Used

Backend	PHP

Libraries	Google Fonts, Font Awesome
